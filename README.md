# \<dile-select>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i dile-select
```

## Usage
```html
<script type="module">
  import 'dile-select/dile-select.js';
</script>

<dile-select></dile-select>
```
